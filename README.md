# Skewed

Generate 3D isometric SVGs.

- Typescript-first package
- Built-in standard 3D shapes
- Utility for isometrically transforming 3D points to 2D ones

<img width="50%" src="./docs/images/isometric.png" style="margin: 0px 25%"/>

```

```
